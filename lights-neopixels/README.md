# lights-neopixels

The neopixel lights come in 1 neopixel, 7 neopixel (jewel), 12 neopixel ring, 24 neopixel ring, and 60 neopixel strip. You can use the same code for all neopixels - you only need to update the number of neopixels. You can chain neopixels together by continuing the alligator clips across multiple neopixels.
